# Streamlit Sentiment Analysis

A Streamlit-based sentiment analysis app that classifies customer feedback as positive, neutral, or negative. The project uses NLP via TF-IDF and Logistic Regression.

## How to run

1. Install the required libraries:

```bash
python -m pip install -r requirements.txt
```

2. Run the Streamlit app:

```bash
streamlit run app.py
```

## Features

- Real-time feedback sentiment classification
- Emoji reaction for positive, neutral, and negative labels
- Built with Streamlit, scikit-learn, and pandas
- Ready for local deployment and extension
